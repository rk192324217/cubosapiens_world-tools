# CuboSapiens Tetris Game

A modern browser-based Tetris game built using HTML5, CSS3, and JavaScript.

## Features

- Fully functional Tetris gameplay
- Easy, Medium and Hard difficulty modes
- Random Tetromino generation
- Score tracking system
- High score storage using Local Storage
- Level progression
- Line clear counter
- Next piece preview
- Pause / Resume support
- Restart functionality
- Game Over screen
- Keyboard controls
- Mobile touch controls
- Responsive design
- Dark mode compatible UI

## Controls

### Desktop

| Key | Action |
|------|---------|
| ← | Move Left |
| → | Move Right |
| ↓ | Soft Drop |
| ↑ | Rotate Piece |
| Space | Hard Drop |
| P | Pause / Resume |

### Mobile

- Left Button → Move Left
- Right Button → Move Right
- Rotate Button → Rotate Piece
- Down Button → Soft Drop

## How to Run

### Using Live Server

1. Open project in VS Code
2. Install Live Server extension
3. Right click `index.html`
4. Click "Open with Live Server"

### Direct Browser

Open `index.html` in any modern browser.
